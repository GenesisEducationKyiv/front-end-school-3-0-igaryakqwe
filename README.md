# Music Platform

## Setup

### Install dependencies

```bash
pnpm install
```

### Run the app
```bash
pnpm run dev
```

## Done

- Creating tracks
- Editing tracks
- Deleting tracks
- Filtering tracks
- Sorting tracks
- Pagination
- Searching tracks

## Additional features

- Bulk deletion
- Optimistic UI updates
- Error handling
- Loading states / skeletons
- Audio player with play/pause/next/prev/volume controls
- Audiowave visualization
- Dark/Light mode
- Testing
